Run a command in a persistent shell.
Default cwd is the session worktree; use workdir instead of cd chains.

Output truncates after ~2000 lines / 50KB; full output is written to a temp file.
Use Read or Grep on that path — do not pipe through head/tail to “fix” truncation.

For finding, reading, or editing files, prefer Glob, Grep, Read, Edit, and Write — not cat, find, grep, or sed in bash.
